# Prabal Agrawal — DevOps Portfolio

Personal portfolio site showcasing production DevOps, platform, and MLOps work.

## Preview locally

```bash
cd prabal-devops-portfolio
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080)

## Deploy (free options)

### GitHub Pages

1. Create a GitHub repo and push this folder
2. Go to **Settings → Pages**
3. Source: **Deploy from branch**
4. Branch: `main` / root
5. Your site will be live at `https://<username>.github.io/<repo-name>/`

### Netlify

1. Drag and drop this folder at [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Or connect the GitHub repo for auto-deploys

### Vercel

1. Import the repo at [https://vercel.com/new](https://vercel.com/new)
2. No build command needed — static site

## Customize

- Projects: edit `js/data.js`
- Colors and layout: edit `css/styles.css`
- Hero and about text: edit `index.html`
